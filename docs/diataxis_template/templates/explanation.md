#### 내부 동작 과정

```mermaid
sequenceDiagram
    participant Input as 입력 데이터
    participant Validator as 검증기

    Input->>Validator: 1. 데이터 검증 요청

        Processor-->>Output: 8. 최종 출력
    else 검증 실패
        Validator-->>Output: 에러 반환
    end
```
