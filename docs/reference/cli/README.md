## 🔍 명령어 카테고리

### 📊 관리 명령어

```bash
# 프로젝트 초기화
cli init [project-name]

# 상태 확인
cli status

# 설정 관리
cli config [set|get] [key] [value]
```

### 🚀 배포 명령어

```bash
# 배포
cli deploy [environment]

# 롤백
cli rollback [version]

# 로그 확인
cli logs [service]
```
