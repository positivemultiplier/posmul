## 🔍 아카이브 이유

### 1. **완료된 작업 보고서**
- 이미 구현 완료된 기능들의 보고서
- 현재 개발에 직접적으로 필요하지 않음
- 히스토리 참조용으로 보관

### 2. **비활성 도메인 문서**
- Study-Cycle 도메인이 현재 비활성 상태
- 관련 문서들이 현재 개발 흐름과 맞지 않음
- 향후 재활성화 시 참조용으로 보관

### 3. **마이그레이션 완료 문서**
- 모노레포 전환, MCP 시스템 도입 등이 완료됨
- 마이그레이션 과정 문서들은 참조용으로만 필요
- 현재 개발 가이드와 중복되는 내용들
