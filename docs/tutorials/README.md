# 📚 Tutorials - 학습을 위한 여행

> **목적**: 안전한 환경에서의 단계별 학습 경험 제공  
> **대상**: 초보자, 처음 접하는 사용자

## 🎯 튜토리얼 특징

- ✅ **단계별 진행**: 명확하고 순차적인 학습 과정
- ✅ **안전한 환경**: 실패해도 괜찮은 연습 환경
- ✅ **구체적 결과**: 각 단계마다 명확한 결과물
- ✅ **완전성**: 처음부터 끝까지 완성된 여정

## 📖 이용 가능한 튜토리얼

### 🚀 기초 입문

- [ ] **PosMul 시작하기** - 첫 번째 예측 게임 참여하기
- [ ] **PMP/PMC 이해하기** - 경제 시스템 기초 체험

### 🏗️ 개발자 입문

- [ ] **개발 환경 설정하기** - 로컬 개발 환경 구축
- [ ] **첫 번째 도메인 구현하기** - DDD 기초 실습

### 🔧 도구 활용

- [ ] **Supabase MCP 시작하기** - 데이터베이스 조작 기초
- [ ] **GitHub MCP 활용하기** - 프로젝트 관리 자동화

## 📝 튜토리얼 작성 가이드

새로운 튜토리얼을 작성할 때는 다음 템플릿을 사용하세요:

```markdown
# 📚 [기능] 시작하기

## 🎯 학습 목표

- [ ] 목표 1
- [ ] 목표 2

## 🛠️ 사전 준비사항

- 도구 설치
- 권한 확인

## 📖 단계별 가이드

### 1단계: [기본 설정] (10분)

### 2단계: [첫 번째 기능] (15분)

### 3단계: [실제 프로젝트] (20분)

## ✅ 완료 확인

- [ ] 체크리스트
```
