# 🔧 How-to Guides - 문제 해결 레시피

## 🎯 How-to Guide 특징

- ✅ **문제 해결 초점**: 명확한 문제와 해결책 제시
- ✅ **목적 지향적**: 특정 목표 달성에 집중
- ✅ **실무적**: 실제 작업 환경 고려
- ✅ **선택지 제공**: 여러 해결 방법 제시

## 📝 How-to Guide 작성 가이드

새로운 가이드를 작성할 때는 다음 템플릿을 사용하세요:

```markdown
# 🔧 How to [구체적 작업]

## 🎯 해결할 문제

- 상황 설명
- 달성 목표

## 🛠️ 사전 요구사항

- 필요 지식
- 도구/권한

## 📖 해결 단계

### 방법 1: [빠른 해결]

### 방법 2: [고급 해결]

## 🔍 문제 해결

- 자주 발생하는 문제
- 해결 방법

## ✅ 완료 체크리스트
```
