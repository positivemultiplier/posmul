## 🔍 설계 원칙 분석 틀

### 1. **철학적 근거**

- 왜 이 원칙이 중요한가?
- 어떤 가치를 추구하는가?

### 2. **실무적 적용**

- 코드에서 어떻게 구현되는가?
- 일상적 개발에 어떤 영향을 주는가?

### 3. **트레이드오프 분석**

- 어떤 것을 포기했는가?
- 그 선택이 합리적인가?

### 4. **진화 가능성**

- 미래에 어떻게 발전할 수 있는가?
- 어떤 제약이나 기회가 있는가?
